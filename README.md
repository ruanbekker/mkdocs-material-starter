# mkdocs-material-starter

mkdocs-material-starter

## Installation

With `virtualenv`:

```bash
virtualenv -p python3 venv
source ./venv/bin/activate
pip install -r requirements.txt
```

## Commands

* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
